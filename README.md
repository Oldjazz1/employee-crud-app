# Employee CRUD App

Простое CRUD-приложение на Spring Boot + PostgreSQL + Docker для управления сотрудниками (создание, чтение, обновление, удаление).

Запуск:
```bash
mvn clean package -DskipTests
docker compose up --build

Swagger UI: http://localhost:8080/swagger-ui.html

Технологии : Java 21 • Spring Boot 3 • Spring Data JPA • PostgreSQL • Docker • Swagger
